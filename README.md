# Login & Registration Web Application
Spring Boot | Thymeleaf | MySQL

## Project Overview

This is a simple authentication-based web application built using Java 17 and Spring Boot.  
The application allows users to register, log in, maintain sessions, and securely log out.

It demonstrates how user authentication works using Spring Boot MVC architecture with database integration.

---

## Features

### User Registration
- Register with First Name, Last Name, Email, and Password
- Email uniqueness validation
- Server-side form validation
- Data stored in MySQL database

### User Login
- Login using registered Email and Password
- Credentials validated from database
- Error message displayed for invalid login

### Session Management
- User details stored in session after login
- Maintains login state while browsing

### Home Page
- Displays:
  - Full Name
  - Email
  - Role
  - Welcome message

### Logout
- Session invalidated
- Redirected to login page
